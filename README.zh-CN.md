# [KOReader Anna's Archive 插件](https://github.com/fischer-hub/annas.koplugin)

简体中文 | [English](./README.md)

**免责声明：** 本插件仅供教育用途。请遵守版权法并自行承担使用责任。

这个 KOReader 插件用于搜索 Anna's Archive，并通过 Anna 的公开镜像页面下载文件。当前实现基于网页抓取，不需要账号登录，也不再支持 Z-library 时代的邮箱、密码、基础 URL、会话或 RPC 配置。

## 功能特色

- 搜索 Anna's Archive 条目。
- 按语言、格式和排序方式筛选结果。
- 直接从结果详情页触发下载。
- 在一个统一的设置界面中管理持久化选项。
- 配置下载目录、镜像策略、自动重试、首选下载源与超时策略。

## 安装方法

1. 下载最新发布中的 `annas.koplugin.zip`。
2. 解压后确认目录名是 `annas.koplugin`。
3. 将该目录复制到设备上的 `koreader/plugins`。
4. 重启 KOReader。

## 使用方法

1. 确保当前位于 KOReader 文件浏览器。
2. 打开“搜索”菜单。
3. 选择“Anna's Archive”。
4. 输入关键词，并按需在搜索对话框中调整排序、语言和格式。
5. 打开一个搜索结果。
6. 点击格式一行开始下载。

如果你想统一配置插件的持久化选项：

1. 打开“搜索”。
2. 选择“Anna's Archive”。
3. 点击“Settings”。

设置界面目前包含：

- 搜索排序、语言与格式
- 下载目录
- 下载完成后自动关闭 Wi-Fi
- Anna 镜像探测策略
- 网络瞬时故障的自动重试次数
- LibGen 下载镜像的首选来源
- 超时策略预设，以及更细的高级超时设置
- 更新检查

说明：

- 不需要账号登录。
- 不再提供 `annas_credentials.lua` 覆盖文件。
- 旧版“推荐图书”“最受欢迎图书”等 Z-library API 入口已移除，因为它们不属于当前 Anna 搜索/下载流程。
- 当前下载仍依赖 Anna 结果页中可用的受支持 LibGen 镜像。

## 手势设置（可选）

如果你想通过手势直接打开搜索：

1. 打开顶部菜单并点击设置图标。
2. 进入“点击与手势” > “手势管理器”。
3. 选择一个手势。
4. 在“一般”分类中勾选 “Anna's Archive search”。

## DNS 建议

只有当 Anna's Archive 在你的网络环境下持续无法访问时，才建议调整 DNS。推荐顺序如下：

1. 先修改路由器 DNS。
2. 只有在无法改动路由器时，才考虑在设备上手动修改系统 DNS。

对 Kindle 和 Kobo 来说，路由器级 DNS 更安全，因为它不需要直接修改阅读器系统文件。

如果你确实需要在设备端修改 DNS，请先阅读英文 README 中的完整操作步骤和风险说明。设备级修改可能会被更新覆盖，某些 Kindle 环境还需要把根文件系统重新挂载为可写。
